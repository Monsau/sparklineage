# ملخص

يوفر هذا المشروع حلاً للتكامل لتتبع نسب البيانات تلقائيًا في Apache Spark، مع عرض مباشر في OpenMetadata. وهو مخصص للفرق الفنية التي ترغب في أتمتة تتبع خطوط أنابيب Spark الخاصة بهم دون تعديل كود التطبيق.
